component ,,pyhoca-cli''.

   o Support for passphrase protected SSH keys.


X2Go Component: pyhoca-cli
Version: 0.4.0.2

  pyhoca-cli (0.4.0.2) RELEASED; urgency=low

      - Add support for unlocking encrypted SSH private key files.
