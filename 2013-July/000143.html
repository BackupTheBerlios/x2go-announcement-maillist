Dear all,

the X2Go project is proud to announce the initial release of a new X2Go
component named ,,x2gomatebindings''.

The project ,,x2gomatebindings'' is a fork of the old X2Go GNOMEv2 Bindings
project (x2gognomebindings). The X2Go MATE Bindings have been tested  
against the MATE desktop environment and they fully integrate into  
mate-session and the mate-file-manager (caja).

   o Add ,,Suspend X2Go Session to the System menu.
   o Icons appear on desktop if client-side folders are shared.
   o Mounted shares can be remove through right-click on the desktop icon.

NOTE: The project is only available for Debian at the moment. Please  
add the MATE project's package archive [1] to your APT configuration  
to make x2gomatebindings (together with mate-desktop-environment)  
installable.

[1] http://wiki.mate-desktop.org/download

MATE desktop session support will be soon added to X2Go Client (with  
4.0.1.1) and PyHoca-GUI (with 0.4.0.5).
