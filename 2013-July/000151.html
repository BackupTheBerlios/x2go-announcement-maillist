Dear all,

the X2Go project is proud to announce a new release of the X2Go
component ,,python-x2go''.

New gains of this version of ,,python-x2go'' are:

   o Support for MATE desktop environment.
   o Catch exception during session startup/resumption when connections fails.


X2Go Component: python-x2go
Version: 0.4.0.6
Status: RELEASE
Date: Sun, 28 Jul 2013 19:42:52 +0200
Changes:
  python-x2go (0.4.0.6) RELEASED; urgency=low
  .
    * New upstream version (0.4.0.6):
      - Use XFCE as desktop environment in example scripts.
      - Make Python X2Go aware of the MATE desktop environment.
      - When resuming session be tolerant towards disrupted connections.
