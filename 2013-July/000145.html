Dear all,

the X2Go project is proud to announce a bugfix release of the X2Go
component ,,x2goserver''.

New gains of this version of ,,x2goserver'' are:

   o Fixes VERSION information when querying x2goversion x2goserver.
   o Fix hard-coded paths in x2goruncommand (relevant for fixing #269).


X2Go Component: x2goserver
Version: 4.0.1.5
Status: RELEASE
Date: Sun, 28 Jul 2013 02:06:47 +0200

Changes:
  x2goserver (4.0.1.5) RELEASED; urgency=low
  .
    * New upstream version (4.0.1.5):
      - Do not hard-code paths to NX code in x2goruncommand anymore. Use
        x2gopath instead. (Fixes: #273).
