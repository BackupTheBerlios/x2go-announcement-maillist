Dear all,

the X2Go project is proud to announce a new release of the X2Go
component ,,x2godesktopsharing''.

New gains of this version of ,,x2godesktopsharing'' are:

   o A cmdline option (--activate-desktop-sharing) that allows sysadmins
     to deploy x2godesktopsharing applets that automatically start the
     sharing mode on applet startup. This feature has been sponsored by
     https://netitwork.net/ (Michael Kromer).
   o Furthermore, the applet is now available in French. Thanks to Frédéric
     Motte for his i18n related work.


X2Go Component: x2godesktopsharing
