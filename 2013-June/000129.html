Dear all,

the X2Go project is proud to announce a new release of the X2Go
component ,,x2goserver''. This release (4.0.0.3) is a component update
for the X2Go Baikal LTS bundle release.

Issues fixed in this version of ,,x2goserver'' are:

   o Some build issues in the Makefile(s) have been addressed
   o The USED_PORTS detection code has been partially reverted
     to an older (but functional) state. This hopefully improves
     reported problems that occur if many clients try to simultaneously
     launch sessions on the same server.
   o Bug closures: #155 (see below)


X2Go Component: x2goserver
Version: 4.0.0.3
Status: RELEASE
Date: Fri, 07 Jun 2013 23:03:20 +0200
Changes:
  x2goserver (4.0.0.3) RELEASED; urgency=low
  .
    * New upstream version (4.0.0.3):
      - Make pre-set DESTDIR and PREFIX traverse through all Makefiles.
        Do not include DESTDIR in Makefile-hard-coded path in
        x2gosqlitewrapper.
      - Partially revert USED_PORTS detection code back to the old-style
        awk-way. (Fixes: #155).
